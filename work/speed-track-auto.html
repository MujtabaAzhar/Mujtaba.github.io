Speed Track Auto

Bilingual Laravel platform for a Saudi car-wash chain — rebuilt from WordPress

Visit live site

THE PROBLEM

The chain's original WordPress site was a static brochure. It couldn't present subscription packages, the loyalty program, or branch details in both Arabic and English — and every real feature lived outside it.

THE STACK

  • Laravel
  • MySQL
  • Arabic/English i18n
  • RTL layout
  • WhatsApp integration

WHAT I BUILT

A ground-up Laravel rebuild of the entire site: fully bilingual Arabic/English with proper RTL rendering, monthly wash subscription packages, the five-visit loyalty program, corporate voucher offerings, and branch pages covering Makkah and Jeddah — with WhatsApp contact built in.

THE OUTCOME

The new site is live at speedtrackauto.com, presenting the chain's packages and loyalty program in both languages and connecting directly to the systems that run the business.
